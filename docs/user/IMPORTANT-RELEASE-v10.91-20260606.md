# 省心PPT v10.91 重要版本记录

日期：2026-06-06

## 本次版本定位

这是一次重要稳定性版本，目标是恢复用户已验收的 PDF 预览体验，同时保留 v10.86 之后已经修复并上线的参数、主题、配图与占位图治理成果。

## 本次已确认修复

1. 恢复旧版 PDF 预览方案
   - 结果页重新恢复 PDF 在线预览。
   - 保留 PPTX 下载能力。
   - 当前方案为：主任务先生成 PDF 用于预览和免费导出，结果页再异步预热第二条 PPTX 任务。

2. 保留后续关键修复，不做整站回退
   - 保留主题参数全链路对齐修复。
   - 保留图片源从搜索网图切到 Pexels 的修复。
   - 保留“极简无图”选项。
   - 保留空白图片占位符修复。

3. 明确当前 PDF 预览的技术结论
   - Gamma 当前公开 API 仍按单次请求只支持一个 exportAs 格式处理。
   - 现阶段要同时保留 PDF 预览和 PPTX 下载，实际链路仍然是两次生成。
   - 这条链路已恢复为当前默认方案，后续等待官方提供更优能力后再迭代。

## 本次计费与积分对齐

1. 当前预览链路会产生两份文稿
   - 一份用于 PDF 预览 / 免费导出 PDF。
   - 一份用于 PPTX 下载。

2. 因此积分扣除已按双生成成本对齐
   - 基础页生成成本按双倍计算。
   - AI 图片成本也按双倍计算。

3. 订阅页换算说明已同步
   - 套餐页数估算已按最新积分消耗口径更新。

## 管理端对齐

1. 管理员账户需直接反映 Gamma 实时总积分。
2. 后台概览和管理员账户视图应以 Gamma key pool 的实时总余额作为核心参考值。

## 后续优化原则

1. 当前版本保持已验收方案，不再折腾预览链路。
2. 等 Gamma 官方提供更稳定、同稿、单次生成可多格式导出的正式能力后，再替换当前双生成方案。
