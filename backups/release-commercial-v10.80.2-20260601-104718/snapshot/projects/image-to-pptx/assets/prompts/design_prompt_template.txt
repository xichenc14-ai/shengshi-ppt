# Design Prompt Template - 设计图提示词模板

## 用途
本模板用于生成 PPT 设计图的 AI 生图提示词。

## 模板结构

```
【构图】16:9比例的PPT整体设计图，展示X张幻灯片的完整布局
【风格】现代简约/商务正式/创意活泼/极简主义
【配色】主色:#XXXXXX 次色:#XXXXXX 背景:#XXXXXX 强调色:#XXXXXX
【布局】标题区域、内容区域、图表区域、页脚区域的位置和大小
【字体】标题字体、正文字体的大小和样式
【元素】具体视觉元素描述
【质量要求】高清晰度、专业的PPT设计感
```

## 风格预设

### 现代简约 (Modern)
- 主色: #2563EB
- 次色: #3B82F6
- 背景: #FFFFFF
- 强调色: #06B6D4
- 字体: 无衬线字体，简洁现代
- 元素: 几何图形、渐变色块、简洁线条

### 商务正式 (Business)
- 主色: #1E3A5F
- 次色: #2D5A87
- 背景: #F8FAFC
- 强调色: #DC2626
- 字体: 衬线字体，稳重专业
- 元素: 图表、数据可视化、商务图标

### 创意活泼 (Creative)
- 主色: #EC4899
- 次色: #F472B6
- 背景: #FDF2F8
- 强调色: #8B5CF6
- 字体: 手写风格或活泼字体
- 元素: 插画、卡通元素、不规则形状

### 极简主义 (Minimalist)
- 主色: #000000
- 次色: #6B7280
- 背景: #FFFFFF
- 强调色: #374151
- 字体: 无衬线字体，极简
- 元素: 大量留白、简洁线条、单色块

## 生成示例

### 示例 1: 产品发布会 PPT
```
【构图】16:9比例的PPT设计图，展示5张幻灯片的完整布局，封面+产品特点+技术参数+应用场景+联系我们
【风格】现代简约
【配色】主色:#2563EB 次色:#3B82F6 背景:#FFFFFF 强调色:#06B6D4
【布局】标题区域位于顶部15%高度，内容区域位于中部70%，页脚区域位于底部10%
【字体】标题: 36pt 加粗无衬线，正文: 18pt 无衬线
【元素】几何图形装饰、渐变色块、产品图片占位符、功能图标
【质量要求】高清晰度，专业的PPT设计感，现代科技感，可用于直接转换为PPTX的设计参考图
```

### 示例 2: 销售业绩汇报 PPT
```
【构图】16:9比例的PPT设计图，展示6张幻灯片的完整布局，封面+华北区域+华东区域+华南区域+总结+感谢页
【风格】商务正式
【配色】主色:#1E3A5F 次色:#2D5A87 背景:#F8FAFC 强调色:#DC2626
【布局】标题区域位于顶部12%高度，内容区域位于中部75%，页脚区域位于底部8%，右侧留出图表区域
【字体】标题: 32pt 加粗衬线，正文: 16pt 衬线
【元素】折线图、柱状图、数据卡片、增长箭头、商务图标、公司Logo占位符
【质量要求】高清晰度，稳重专业的商务风格，数据可视化清晰，可用于直接转换为PPTX的设计参考图
```

## 注意事项

1. **16:9 比例**: 始终使用 16:9 (1920x1080 或等比) 作为设计比例
2. **清晰布局**: 每页布局需要有明显的视觉层次
3. **配色一致性**: 同一 PPT 的所有页面应使用一致的配色方案
4. **字体可读性**: 字体大小需要保证在小图预览时也能看清
5. **元素可提取性**: 设计图中的图标、背景等元素应该可以被裁剪提取

## 后续处理

生成的设计图可用于：
1. AI 视觉分析 → 提取布局信息
2. 素材提取 → 裁剪图标、背景等元素
3. 直接参考 → 作为 PPTX 生成的布局模板
