# 办公室AI课程第三课：AI写作与智能体应用

## 内容结构（4大模块）

---

### 模块 1：AI写作基础工具（10分钟）

#### 秘塔写作猫
- **定位**：专注中文写作的AI助手
- **核心功能**：错别字检查、语法纠错、文章改写润色、敏感词过滤
- **使用场景**：公文撰写、论文校对、商业邮件
- **亮点**：支持WPS/Word插件，写作时实时检测

#### Kimi
- **定位**：长文本处理专家
- **核心功能**：超长文档（20万字）阅读理解、摘要提取、多语言翻译
- **使用场景**：合同审阅、文献综述、长报告整理
- **亮点**：支持上传PDF/Word/PPT，直接提问文档内容

#### 讯飞听见
- **定位**：语音转文字与AI辅助写作
- **核心功能**：会议录音转写、AI摘要生成、待办事项提取
- **使用场景**：会议纪要、访谈记录、演讲整理
- **亮点**：支持多种方言，转写准确率98%+

### 模块 3：专业写作工作流（10分钟）

#### 标准五步流程
```
选题 → 大纲 → 初稿 → 校对 → 发布
```

#### 选题阶段
- **工具**：Kimi、DeepSeek
- **方法**：输入行业关键词，AI推荐选题方向
- **技巧**：要求AI给出3个不同角度的选题方案

#### 大纲阶段
- **工具**：Coze扣子、智谱清言
- **方法**：输入主题，自动生成文章结构
- **技巧**：指定风格（专业/轻松/营销）和字数要求

#### 初稿阶段
- **工具**：DeepSeek、豆包、文心一言
- **方法**：投喂大纲，AI逐段扩展内容
- **技巧**：要求分步骤生成，每段单独优化

#### 校对阶段
- **工具**：秘塔写作猫、讯飞听见
- **方法**：全文错别字检测、语病修改、敏感词过滤
- **技巧**：开启AIGC检测，确保文章"去AI味"

#### 发布阶段
- **工具**：各平台原生编辑器
- **方法**：一键复制到公众号/知乎/小红书
- **技巧**：根据平台特性微调标题和开头

### 模块 4：内容创作实战（10分钟）

#### 实战一：周报生成
**提示词模板**：
```
请根据以下工作记录，帮我生成一份周报：
[粘贴工作内容]
要求：
- 结构清晰，分成【本周完成】【问题与解决】【下周计划】
- 数据量化成果
- 语言简洁专业
```

#### 实战二：营销文案编写
**提示词模板**：
```
请为以下产品写3条不同风格的推广文案：
产品：[产品名称+卖点]
目标人群：[受众描述]
要求：
- 文案1：种草风格，适合小红书
- 文案2：专业风格，适合知乎
- 文案3：促销风格，适合电商详情页
```

#### 实战三：智能体创建（Coze扣子）
**操作步骤**：
1. 登录 coze.cn，点击"创建智能体"
2. 选择"知识库"模式，上传产品文档
3. 配置开场白和引导问题
4. 调试对话效果
5. 一键发布到"豆包"或"微信客服"

**提示词示例**：
```
你是一位专业的[职位]，负责回答用户关于[产品/服务]的问题。
回答风格：专业、耐心、有条理。
遇到无法回答的问题，礼貌引导用户联系人工客服。
```
