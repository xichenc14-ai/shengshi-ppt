# 省心PPT 全站视觉升级方案（v10.51）

## 1. 目标
- 提升“专业 + 省心”的品牌辨识度，突出 PPT 设计平台属性。
- 缩短用户从进入首页到开始生成的决策时间。
- 统一全站视觉与文案风格，减少中英混杂与认知负担。
- 提升关键转化：开始生成点击率、生成完成率、下载率。

## 2. 品牌与信息层级
- 品牌主标语（首页首屏固定）：`省心PPT，让您真省心！`
- 副标语：`从需求到成稿，全流程AI协同，30秒完成专业演示初稿`
- 首页首屏信息优先级：
  1) 品牌与主承诺（省心、专业、快）
  2) 输入框与模式选择（核心动作）
  3) 场景入口与示例（降低输入门槛）
  4) 过程可信度（生成步骤、用户评价、数据）

## 3. 视觉系统（Design System）
- 主色：保留品牌紫（`#5B4FE9`）作为唯一主行动色。
- 辅色：成功绿、警示橙、信息蓝，仅用于状态提示，不抢主色。
- 字体层级：
  - H1：30/36，粗体，品牌承诺
  - H2：22/28，粗体，模块标题
  - Body：14/22，常规说明
  - Caption：12/18，辅助提示
- 圆角与阴影：
  - 全站卡片圆角统一 `12px`（重要弹窗可 `16px`）
  - 阴影统一 2 档：`card`、`floating-action`
- 图标风格：统一线性图标 + 少量语义 emoji（仅用于强化，不作为主表达）。

## 4. 首页改版建议（重点）
- 首屏改为“一屏可完成输入+模式选择+开始生成”：
  - 顶部：品牌名 + 标语
  - 中部：大输入框（可拖拽上传）
  - 下部：专业模式 / 省心定制双入口卡
  - 主按钮：`开始生成PPT`
- 新增“设计感展示带”（首屏下方）：
  - 3 组示例缩略图：商务汇报 / 教学课件 / 路演提案
  - 每组显示“主题风格 + 页数 + 生成耗时”
- 首页底部模块顺序优化：
  1) 生成流程（3 步）
  2) 真实案例与口碑
  3) 套餐权益
  4) 常见问题

## 5. 核心流程页优化
- 输入页：
  - 高级选项默认折叠；展开后使用分组标题（主题/文本处理/配图/页数）。
  - 所有用户可见文案中文化，去掉技术英文暴露。
- 大纲编辑页：
  - “下一步：生成PPT”按钮固定显示（加载时禁用，不隐藏）。
  - 页面底部增加弱提示：`可拖拽排序、可直接改标题与要点`。
- 生成中页：
  - 强化“可离开页面，稍后自动恢复”提示。
  - 进度文案统一短句，减少技术术语。
- 结果页：
  - 默认自动加载在线预览。
  - 主行动保持单一：`下载PPTX`。

## 6. 文案规范（全站）
- 统一中文术语：
  - `preserve` -> `保持原样`
  - `condense` -> `总结提炼`
  - `generate` -> `扩充文本`
  - `credits` -> `积分/费用`
- 技术编码（如 themeAccent）只在内部使用，不在用户界面直接展示。

## 7. 动效与体验
- 动效原则：短、轻、少，避免花哨拖慢感知。
- 按钮反馈：hover、active、disabled 三态清晰。
- 骨架屏：用于预览区域与历史列表，避免“空白等待”。

## 8. 分阶段落地（建议两周）
- 第1阶段（2-3天）：文案中文化 + 首页首屏优化 + 按钮稳定性修复。
- 第2阶段（3-4天）：视觉系统变量化（颜色、圆角、阴影、间距）+ 组件统一。
- 第3阶段（3-4天）：案例展示带 + 结果页布局优化 + 回归测试。
- 第4阶段（1-2天）：数据复盘（点击率、完成率、下载率）并微调。

## 9. 验收指标
- 首页“开始生成”点击率提升 ≥ 15%
- 大纲页到生成页转化率提升 ≥ 10%
- 结果页下载率提升 ≥ 8%
- 用户反馈中“看不懂/太技术化”类问题下降 ≥ 50%
