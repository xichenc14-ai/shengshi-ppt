# 省心PPT 商业化最终验收清单（v10.80.2）

更新时间：2026-06-01

## A. 环境与支付（P0）

1. 环境就绪检查
- 执行：`npm run -s env:commercial`
- 判定：`Overall: READY`
- 操作参考：`docs/user/COMMERCIAL-ENV-INJECTION-SOP-v10.80.2.md`

2. 发布前体检
- 执行：`npm run -s preflight:commercial`
- 判定：`PASS`

3. 审计留档
- 执行：`npm run -s audit:commercial`
- 判定：生成报告结论 `PASS`

4. 支付联调验收（生产等价环境）
- 创建订单成功（返回支付链接/二维码）
- 回调验签通过（微信/支付宝）
- 回调金额与订单金额一致
- 订单状态 `pending -> completed`
- 会员开通成功、积分到账成功

## B. 质量门禁（P0）

1. `npm run -s lint`：无 error
2. `npm run -s test:run`：全部通过
3. `npm run -s build`：通过

## C. 合规与安全（P0）

1. 协议页面可访问
- `/privacy`
- `/terms`
- `/service-terms`

2. 协议入口可点击
- 登录弹窗
- 支付弹窗
- 页脚链接

3. 安全头与限流
- CSP/HSTS/Frame/Content-Type 等响应头已开启
- 登录、支付、生成接口限流生效

## D. 手机端 UI 验收（P1）

断点检查：`375x812`、`390x844`、`430x932`

1. 首页主题色卡
- 每屏不再出现“单卡巨大占版”问题
- 卡片与文字可读，点击无误触

2. 生成参数弹窗主题选择器
- 分类区与主题区在手机端为紧凑网格
- 选中态可辨识（边框/底色）

3. 支付/登录弹窗
- 不超出可视区域
- 关键 CTA 可见且可点击

## E. 上线放量建议（P1）

1. 灰度发布 10%-30%-100%
2. 观察 30 分钟核心指标：
- 支付成功率
- 支付回调成功率
- 生成成功率
- 导出成功率
3. 若异常，回滚到最近备份版本并保留审计报告
