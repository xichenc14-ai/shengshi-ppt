# UI Style Lock (v10.60)

更新时间：2026-05-20

## 锁定结论

从 **v10.60** 起，省心PPT 全站默认视觉风格锁定为：

- 紫蓝渐变主色（品牌核心）
- 玻璃质感卡片（glassmorphism）
- 海报级首屏 + 强层次分区
- 轻动效（进度、浮层、光扫、状态反馈）

## 允许调整

- 文案、数据、按钮文案、信息层级
- 间距与排版微调（不改变整体视觉语言）
- 交互动效节奏优化（不改变风格方向）

## 不允许的改动（除非业务方明确批准）

- 更换主色系到非紫蓝系
- 把海报风改回极简素白风
- 删除玻璃层级与核心动效语言
- 未评审情况下进行大改版（布局范式重做）

## 执行备注

后续迭代在该基线内演进，优先做体验优化与模块质量提升，不做无审批的大外观变更。
