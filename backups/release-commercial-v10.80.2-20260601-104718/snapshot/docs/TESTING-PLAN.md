# 省心PPT 测试方案 v1.0

> **版本：** v1.0
> **创建日期：** 2026-04-24
> **项目：** 省心PPT（demo v9.5.1）

---

## 一、测试分级

### P0 - 冒烟测试（每次发版必测）
核心功能是否可用，验证基本流程跑通。

### P1 - 回归测试（每周/重大更新）
已验证功能的稳定性，防止回归。

### P2 - 功能测试（新功能上线）
新功能完整测试。

### P3 - 边界/性能测试（季度）
压力测试、边界条件。

---

## 二、P0 冒烟测试项目

### 2.1 首页访问
- [ ] 页面加载 < 3秒
- [ ] UI 布局正确，无错位
- [ ] 响应式适配（PC/移动端）
- [ ] 无 console 报错

### 2.2 专业模式生成
- [ ] 输入主题 → 生成大纲 → 编辑大纲 → 确认生成
- [ ] 生成状态轮询正常
- [ ] 生成完成后可预览
- [ ] 错误状态正确显示

### 2.3 省心模式生成
- [ ] 输入主题 → 一键生成
- [ ] 生成结果正确展示
- [ ] 可切换到专业模式重新生成

### 2.4 附件上传
- [ ] PDF 上传 → 内容提取 → 可编辑
- [ ] Word 上传 → 内容提取 → 可编辑
- [ ] 上传失败有错误提示

### 2.5 主题切换
- [ ] 主题列表正确展示（7个色系）
- [ ] 切换主题后预览更新
- [ ] 生成时主题正确传递给 Gamma API

### 2.6 预览功能
- [ ] 幻灯片渲染正确
- [ ] 左右翻页正常
- [ ] 页码显示正确

### 2.7 导出功能
- [ ] PPTX 导出 → 文件可下载
- [ ] PDF 导出 → 文件可下载
- [ ] 导出文件可正常打开
